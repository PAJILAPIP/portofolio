Portofolio Muhammad Fazil Afif

Website Portofolio pribadi untuk menampilkan informasi diri, lini masa akademis dan pengalaman, serta koleksi proyek (portofolio) dari Muhammad Fazil Afif. Dibangun menggunakan HTML, CSS kustom, dan kerangka kerja Bootstrap 5.

Struktur File
Proyek ini terdiri dari file-file utama berikut:

* index.html: Halaman beranda (Home).
* about.html: Halaman tentang saya (About), berisi detail profil, visi, misi, dan lini masa (timeline) akademis, pengalaman, dan keahlian menggunakan komponen Accordion.
* portofolio.html: Halaman Portofolio, menampilkan daftar proyek dengan fitur filter berdasarkan kategori (Java, Web Dev, Mobile App) dan modal untuk detail proyek.
* contact.html: Halaman Kontak, berisi informasi kontak dan formulir untuk menghubungi.
* style.css: Berkas Cascading Style Sheets yang berisi styling kustom untuk seluruh halaman.
* filter.js: Berkas JavaScript untuk mengaktifkan fungsionalitas filter pada halaman portofolio.
* validation.js: Berkas JavaScript untuk mengimplementasikan validasi formulir dan logika pengiriman pada halaman kontak.

Teknologi yang Digunakan

* HTML5
* CSS3
* Bootstrap 5.3.8: Digunakan untuk tata letak responsif, navigasi, card, accordion, carousel, dan modal.
* Bootstrap Icons 1.13.1: Digunakan untuk ikon-ikon.
* JavaScript (Vanilla JS): Digunakan untuk fungsionalitas kustom (filter portofolio dan validasi formulir).

Fitur Utama

1. Desain & Tampilan
* Responsif: Menggunakan Bootstrap, website dioptimalkan untuk berbagai ukuran layar.
* Dark Theme: Menggunakan skema warna gelap (kode CSS menunjukkan #222831 dan #393E46) dengan aksen biru (#00ADB5) untuk tampilan modern.
* Glassmorphism: Efek transparan/blur ringan pada navigasi (.glass-bg).
* Hover Effects: Efek transisi halus pada kartu portofolio dan ikon sosial media.

2. Halaman Home (index.html)
* Sapaan utama dan judul.
* Bagian fitur/minat (.feature-section) dalam bentuk card.
* Galeri foto menggunakan komponen Carousel.

3. Halaman About (about.html)
* Foto profil melingkar dengan border (.foto-bingkai).
* Detail Visi & Misi.
* Lini masa informasi yang disajikan dalam komponen Accordion (Academic Timeline, Experience Timeline, Achievements Timeline, Skills).

4. Halaman Portofolio (portofolio.html)
* Fitur Filter Proyek (Filter.js) untuk menyaring proyek berdasarkan kategori (All, Java, Web Dev, Mobile App).
* Proyek ditampilkan dalam komponen Card yang dapat diklik untuk memunculkan detail melalui Modal.

5. Halaman Contact (contact.html)
* Informasi kontak lengkap (Alamat, Telepon, Email, Media Sosial).
* Peta lokasi (iframe).
* Formulir Kontak dengan validasi klien-side (Validation.js) menggunakan fitur Bootstrap needs-validation.
    * Validasi memastikan semua field wajib terisi.
    * Menampilkan pesan sukses atau gagal menggunakan komponen Alert.
    * Tombol Submit dinonaktifkan hingga formulir valid.

6. Navigasi & Footer
* Sticky Navbar: Navigasi tetap berada di bagian atas saat menggulir (scroll).
* Footer: Berisi bagian "About Me", "Quick Links", dan "Follow Me" dengan ikon sosial media.
